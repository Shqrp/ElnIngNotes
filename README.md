## Usage

You can download the notes' PDF files in the Releases tab. If I still haven't released the note you need, you just clone the repo and compile your desired file. For example, to compile Calculus I notes, you have to run the following commands.
```
git clone https://github.com/Shqrp/ElnIngNotes
cd ElnIngNotes
typst compile Dispense/Analisi\ Matematica\ I.typ
```
The requested PDF will be available in the `Dispense/` directory. This will also let you modify the documents to your liking.

The flashcards were created and studied through [Anki](https://apps.ankiweb.net/). To import them into Anki, you just press `File->Import` or the key combination `CTRL-Shift-I` and select the desired `.txt` file.
